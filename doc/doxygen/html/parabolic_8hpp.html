Detailed Description

Redirection hook and placeholder for the Parabolic IGR subsystem.

Note
The parabolic artificial viscosity method is implemented as member functions of the main Solver class:
  • Solver::compute_igr_parabolic_rhs()
  • Solver::compute_entropic_pressure()

Their corresponding declarations are located in solver.hpp, and implementations are located in parabolic.cpp. This file acts as a placeholder to preserve compilation unit boundaries.
