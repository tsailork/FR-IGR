Detailed Description

Represents a dynamic line or parabola solid boundary.

Defined as f(xi) = a*xi^2 + b*xi + c along X or Y, where coefficients interpolate linearly with dynamic time parameter q: a(q) = (1-q)*a0 + q*a1 b(q) = (1-q)*b0 + q*b1 c(q) = (1-q)*c0 + q*c1
