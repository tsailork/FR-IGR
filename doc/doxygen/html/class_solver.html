Detailed Description

Central solver class governing numerical integration, multi-block communication, and limiters.

Manages the global high-order FR formulation, coordinating operations like Riemann fluxes, artificial viscosity via Isotropic Gradient Regularisation (IGR), and explicit SSP-RK3 time-marching for the compressible 2D Euler/Navier-Stokes equations.

Asserts the physical validity of the global thermodynamic state.

Scans all degrees of freedom to ensure density ( ) and pressure ( ) remain positive, and that no NaN values have been introduced. If a stability violation is found, it terminates the execution and dumps the failing state vector to stderr.

◆ sweep_x()

void Solver::sweep_x ( )

Perform Flux Reconstruction 1D inviscid sweep in the X coordinate direction.

Perform the X-direction Flux Reconstruction sweep.

Accumulates the flux divergence into the RHS vector. Utilizes the FR (Correction Procedure via Reconstruction) method with tensor-product bases and Radau correction polynomials to restore high-order accuracy up to the element boundaries.

Data Structures & Indexing:

  • U(v, ey, ex, iy, ix): The global conserved state array. Read-only in this pass.
  • RHS(v, ey, ex, iy, ix): The global explicit right-hand side array. This function accumulates the flux divergence and interface corrections into RHS via subtraction (-=).
  • sigma_field[get_flat_idx(ey, ex, iy, ix)]: The global scalar entropic pressure field.
  • basis.l_L, basis.l_R: 1D arrays of size N_PTS used to extrapolate solution points to the left/right faces.
  • basis.dgl, basis.dgr: 1D arrays of size N_PTS containing the derivatives of the Radau correction polynomials.

Assumptions:

  • The global arrays are pre-allocated and RHS is zeroed or holds previous partial accumulations.
  • The memory access pattern is designed to be OpenMP thread-safe by assigning disjoint rows (ey) to different threads.
Note
Calls Solver::solve_riemann for inter-element fluxes and Solver::get_flux_pointwise for internal fluxes.

void Solver::sweep_y ( )

Perform Flux Reconstruction 1D inviscid sweep in the Y coordinate direction.

Perform the Y-direction Flux Reconstruction sweep.

Accumulates the flux divergence into the RHS vector. Utilizes the FR (Correction Procedure via Reconstruction) method with tensor-product bases and Radau correction polynomials.

Data Structures & Indexing:

  • U(v, ey, ex, iy, ix): The global conserved state array. Read-only in this pass.
  • RHS(v, ey, ex, iy, ix): The global explicit right-hand side array. This function accumulates the flux divergence and interface corrections into RHS via subtraction (-=).
  • sigma_field[get_flat_idx(ey, ex, iy, ix)]: The global scalar entropic pressure field.
  • basis.l_L, basis.l_R: 1D arrays of size N_PTS used to extrapolate solution points to the bottom/top faces.
  • basis.dgl, basis.dgr: 1D arrays of size N_PTS containing the derivatives of the Radau correction polynomials.

Assumptions:

  • The global arrays are pre-allocated and RHS contains the partial accumulation from sweep_x.
  • The memory access pattern is designed to be OpenMP thread-safe by assigning disjoint columns (ex) to different threads.
Note
Completes the divergence operator application started by Solver::sweep_x.
