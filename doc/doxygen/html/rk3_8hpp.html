Detailed Description

Redirection hook and placeholder for the temporal integrator subsystem (SSP-RK3).

Note
The Strong Stability Preserving Runge-Kutta 3rd order time-stepper is implemented as member functions of the main Solver class:
  • Solver::step_rk3(double dt)

The corresponding declaration is located in solver.hpp, and its implementation is located in rk3.cpp. This file acts as a placeholder to preserve compilation unit boundaries.
