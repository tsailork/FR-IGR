◆ apply()

void IC::apply ( Solver & solver)

Applies the chosen initial condition to the state across all blocks.

Evaluates the initialization configuration (e.g., Riemann 2D configuration, Blast wave, Isentropic Vortex) and assigns the exact physical values to all Gauss-Legendre quadrature points.

Parameters
[in,out]solverThe main solver instance whose blocks will be initialized.
See also
Parameters::IC_TYPE
Note
Overwrites any existing state unless an immersed boundary (IB) mask prevents it.

Evaluates a sigmoid function for initial condition smoothing.

Smooths discontinuous initial states (like shock waves) over a characteristic width to prevent immediate Gibbs-like oscillations in high-order polynomial basis representations.
