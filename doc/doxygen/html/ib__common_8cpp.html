Detailed Description

Implementation of Immersed Boundary (IB) mask generators and signed distance functions.

Contains implementations for various immersed boundary shapes including circles, NACA airfoils, quadrilaterals, and parabolas. It calculates the signed distance functions (SDF) and solid masks (both sharp and regularized Heaviside) used in the Shifted Boundary Method (SBM) and Volume Penalty Method (VPM).
