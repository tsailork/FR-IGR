◆ apply_entropy_limiter()

Limiters::LimiterStats Limiters::apply_entropy_limiter ( Solver & solver)

Apply the entropy minimum preservation limiter to the active solver grid.

Scans each block element, identifies the local specific entropy floor, and scales polynomial coefficients toward the cell average where necessary using bisection.

Parameters
[in,out]solverThe active solver instance whose state fields will be limited.
Returns
LimiterStats containing the count of modified elements and average scaling parameter.

◆ apply_positivity_limiter()

Limiters::LimiterStats Limiters::apply_positivity_limiter ( std::vector< Cell * > & cells,
const Basis & basis,
const Parameters & p )

Enforce positive density and pressure fields at all physical degrees of freedom.

Checks density and pressure values at both internal solution points and face-extrapolated interface points. If values drop below the defined floor, scales the element's local polynomial coefficients around the cell average to recover positivity while preserving mass.

Parameters
[in,out]cellsFlat vector of computational cells.
[in]basisThe high-order polynomial basis definition.
[in]pGlobal simulation parameters defining positivity floors.
Returns
LimiterStats containing the count of modified elements and average scaling parameter.
