Limiters::LimiterStats Limiters::apply_entropy_limiter ( Solver & solver)

Apply the entropy minimum-preserving limiter to the entire solver domain.

Prevent spurious entropy undershoots by scaling toward the cell average. The entropy floor is the minimum over the cell and its face neighbours.

Enforces specific entropy bounds to maintain positivity and stability in high-order FR methods without introducing excessive numerical dissipation.

Parameters
[in,out]solverThe active Solver context
Returns
LimiterStats Statistics containing the number of cells limited and average theta scaling factor

Limiters::LimiterStats Limiters::apply_positivity_limiter ( State & U,
const Basis & basis,
const Parameters & p )

Apply the Zhang-Shu positivity-preserving limiter.

Enforce ρ ≥ ε and p ≥ ε at every solution point by scaling toward the cell average. Each element is independent.

This limiter maintains the physical admissibility of the solution by ensuring that density and pressure remain positive (or above a small threshold ). It operates by linearly scaling the high-order polynomial towards the cell average.

Data Structures & Indexing:

  • U(v, ey, ex, iy, ix): The global conserved state array. Read and mutated in-place.

Assumptions:

  • Elements are completely independent. Limiting relies strictly on the local element's interior solution points to compute the cell average and the minimum value. No neighbor ghost states are accessed, guaranteeing perfect OpenMP thread safety across elements.
Parameters
UThe state to be limited (modified in place).
basisThe polynomial basis (used for computing cell averages and extrapolations).
pThe solver parameters, containing the positivity threshold POS_LIMITER_EPS.
Returns
LimiterStats Statistics containing the number of cells limited and average scaling factor .
