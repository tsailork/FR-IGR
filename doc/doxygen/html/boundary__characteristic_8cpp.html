Construct a characteristic ghost state using 1D Riemann invariants.

Calculates a ghost boundary state normal to the computational interface using one-dimensional characteristic variables and upwinded wave propagation:

  • Riemann invariant
  • Riemann invariant Upwinds invariants, entropy, and tangential velocity according to local wave speeds. This is essential for non-reflecting far-field boundary conditions.
