Computes the mask value chi (solid indicator) for a cylinder/circle shape.

Computes the solid volume fraction mask for a circle.

Parameters
xPhysical X coordinate of the query point
yPhysical Y coordinate of the query point
center_xCircle center X coordinate
center_yCircle center Y coordinate
radiusCircle radius
sharpTrue if using sharp mask, false if using smoothed Heaviside
smooth_widthInterface smoothing width factor (in units of grid spacing)
dxElement size in X
dyElement size in Y
Returns
Indicator value chi in [0, 1] (1 = fully inside solid, 0 = fully in fluid)

Generates an indicator function where 1 indicates solid and 0 indicates fluid. If a smoothed interface is requested, it uses a regularized Heaviside function to smear the interface over a specified width. This regularization is crucial for stability in high-order FR Volume Penalty Methods (VPM).

Parameters
xX coordinate of the query point
yY coordinate of the query point
center_xX coordinate of the circle center
center_yY coordinate of the circle center
radiusRadius of the circle
sharpTrue to use a sharp Heaviside step, false for a smoothed transition
smooth_widthSmoothing width multiplier (epsilon = smooth_width * h)
dxElement size in x-direction
dyElement size in y-direction
Returns
Indicator value

Accurately models a NACA 4-digit airfoil by rotating the query point to align with the chord line and evaluating the thickness and camber equations.
