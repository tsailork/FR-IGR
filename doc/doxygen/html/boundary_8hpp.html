Detailed Description

Declarations for modular boundary condition utilities and ghost state generators.

This file contains definitions for constructing ghost states at the domain boundaries. Ghost states are utilized to impose various physical boundary conditions such as viscous no-slip walls, characteristic boundaries, and prescribed pressure boundaries. These are fundamental for solving the 2D Euler equations with the Flux Reconstruction (FR) method.
