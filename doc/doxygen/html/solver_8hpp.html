Detailed Description

Main Solver class and grid block structures for the FR-IGR simulation engine.

Declares the core Solver class which manages multi-block grid domains, coordinates numerical sweeps (inviscid and viscous), handles artificial viscosity smoothing (IGR), and advances the solution in time using SSP-RK3.
