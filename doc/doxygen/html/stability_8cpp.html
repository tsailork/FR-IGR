Detailed Description

Stability error detection and CFL dynamic stability bounds computation.

This file contains critical routines to guarantee the stability of the high-order Flux Reconstruction solver. It enforces strict positivity on density and pressure, halting execution if non-physical values arise. Additionally, it dictates the dynamic time-stepping limits based on local wave speeds.
