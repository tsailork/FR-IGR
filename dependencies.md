# FR-IGR Solver Dependencies

To compile, run, and visualize results from the FR-IGR solver, the following software and libraries are required:

## 1. Compiler and Build Tools
*   **C++ Compiler**: `g++` supporting the **C++17** standard or later.
*   **Build System**: `GNU Make`.
*   **Shell Environment**: `bash` (required for running `run.sh`).

## 2. Libraries
*   **OpenMP**: Required for multi-core parallelization. This is typically included with `g++` (enable via `-fopenmp`).

## 3. Visualization and Post-Processing
*   **ParaView**: The primary tool for visualizing the `.pvd`, `.vtm`, and `.vts` output files generated by the solver.
*   **Gnuplot** (Optional): Used by some scripts (`fullplot.gp`, `liveplot.gp`) for 1D or residual plotting.

## 4. Hardware Requirements
*   **Multi-core CPU**: The solver is optimized for parallel execution via OpenMP. The number of threads can be configured in `inputs.dat`.

---
*Note: This solver is developed on Windows using the WSL environment and standard GNU tools.*
